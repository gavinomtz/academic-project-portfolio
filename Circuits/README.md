# Electric Circuits Laboratory

This repository contains laboratory reports, analysis files, simulations, and documentation for my Electric Circuits lab course. The labs in this repository demonstrate the application of core circuit analysis principles through measurement, simulation, instrumentation, and theoretical verification. Each exercise focuses on bridging mathematical circuit models with real measured behavior.

## Lab Topics Completed

- **Lab 0:** The Digital Multimeter, Breadboard, and Lab Safety
- **Lab 1:** Series–Parallel Circuits and Kirchhoff Laws
- **Lab 2:** PSpice Simulation & MATLAB
- **Lab 3:** Nodal Analysis
- **Lab 4:** Mesh Analysis
- **Lab 5:** Superposition Principle
- **Lab 6–7:** Thevenin and Norton Equivalent Circuits
- **Lab 8:** Oscilloscope & Function Generator
- **Lab 9:** RC and RL Time Constant Measurements
- **Lab 11:** Sinusoidal Signals & Phasors

## Tools & Equipment Used

- Digital Multimeter (DMM)
- Oscilloscope
- Function Generator
- DC Power Supply
- PSpice / LTspice simulations
- MATLAB (signal analysis / numeric computation)

## Repository Purpose

This repository is used for:

- Maintaining version control for lab write-ups and analysis files
- Documenting experimental results and comparisons with theory
- Organizing simulations and data files in a reproducible format
