### [Electric Circuits I](./Circuits)
**Fall 2025 | Circuit Analysis & Simulation**

Hands-on laboratory course in DC/AC circuit analysis using professional engineering simulation tools and computational verification methods.

**Core Topics Covered:**
- **DC Circuit Analysis** - Ohm's Law, Kirchhoff's laws, resistive networks
- **Circuit Simulation** - PSpice schematic capture and analysis
- **Computational Verification** - MATLAB matrix operations for circuit equations
- **Nodal Analysis** - Systematic KCL application for node voltages
- **Mesh Analysis** - Systematic KVL application for loop currents
- **Multi-Tool Workflows** - Integrating simulation, computation, and hand calculations

**Laboratory Focus:**
- Professional EDA tool proficiency (OrCAD PSpice)
- Engineering computation with MATLAB
- Cross-validation methodologies (analytical vs simulated vs computed)
- Technical documentation and report writing
- Systematic problem-solving approaches

**Key Skills Demonstrated:**
- Circuit modeling and simulation
- Matrix-based equation solving
- Engineering software proficiency
- Validation and verification practices
- Technical report writing
- Multi-tool integration workflows

**Technologies:** OrCAD PSpice (circuit simulation), MATLAB (numerical computation), Word (technical documentation)

**Laboratory Experiments:** Progressive complexity from simple resistor networks to multi-node analysis requiring systematic problem-solving methods
