### Computer Science I - Detailed Overview

**Course Focus:** Introduction to programming using C++ with emphasis on problem-solving, algorithm design, and object-oriented principles.

**Learning Objectives:**
- Understand fundamental programming concepts (variables, data types, operators)
- Master control structures (if/else, loops, switch statements)
- Design and implement functions with proper parameter passing
- Apply object-oriented programming concepts (classes, objects, methods)
- Process files for input/output operations
- Develop programs that are well-documented and maintainable
- Debug and test programs systematically

**Project Types:**
1. **Calculators** - Geometry calculations, rational number operations, statistical analysis
2. **Converters** - Currency exchange, telephone keypad encoding, unit conversions
3. **Analyzers** - File searching, election results, hotel occupancy, grade distributions
4. **Simulators** - Magic 8-ball predictions, checkerboard generation, game logic
5. **Database Systems** - Movie search with multiple criteria, data filtering

**Skills Progression:**
- **Weeks 1-4:** Basic I/O, variables, arithmetic operations
- **Weeks 5-8:** Conditionals, loops, functions
- **Weeks 9-12:** Arrays, vectors, file operations
- **Weeks 13-16:** Object-oriented programming, classes, inheritance

### Electric Circuits Laboratory - Detailed Overview

**Course Focus:** Hands-on experience with DC/AC circuit analysis using industry-standard simulation tools and computational verification methods.

**Learning Objectives:**
- Analyze resistive DC circuits using Ohm's Law and Kirchhoff's laws
- Use PSpice for circuit simulation and verification
- Apply MATLAB for solving systems of circuit equations
- Compare analytical, simulated, and computational results
- Document engineering work with professional standards
- Develop multi-tool workflows for circuit analysis

**Laboratory Methods:**
1. **Hand Calculations** - Analytical solutions using circuit laws
2. **PSpice Simulation** - Virtual circuit testing and measurement
3. **MATLAB Computation** - Matrix-based equation solving
4. **Cross-Validation** - Comparing results across all three methods
5. **Technical Reporting** - Documenting procedures, results, and analysis

**Skills Progression:**
- **Labs 1-3:** Basic resistor networks, series/parallel combinations
- **Labs 4-6:** Nodal analysis, mesh analysis, Thevenin/Norton
- **Labs 7-9:** AC circuits, impedance
- **Labs 10-11:** Frequency response, phasors
